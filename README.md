TRPG-Character-Sheet
===

A project for the TRPG player and GM who suffering from managing character sheets.

- Main Web
    - Groups
        - Group1

        - Call of Cthulu
        - Sword World
        - ......
## Using

- Docker
- Nodejs
- MongoDB
- MacOS
